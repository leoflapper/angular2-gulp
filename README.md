# Angular 2 + GULP boilerplate #
Experimental Angular 2 + GULP boilerplate.

## Configuration
- **Platform:** node
- **Framework**: express
- **CSS Framework**: bootstrap
- **CSS Preprocessor**: sass
- **JavaScript Framework**: angularjs 2
- **Build Tool**: gulp

## Installation ##
1. Run ``npm install``
2. Run ``gulp watch`` or ``node server.js``
